üìù Deep Learning Notes

Programming is the process of giving instructions to a computer to perform tasks. These instructions are written in a programming language like Python, JavaScript, or C++.

1. Variables

Variables are used to store data. You can name them anything (no spaces), and they hold values like text or numbers.

let age = 25; // age is a variable storing a number

2. Data Types

Different types of data can be stored in variables:

  • String: Text ‚Äì e.g., "Hello"
  • Number: Numeric ‚Äì e.g., 10
  • Boolean: True or False ‚Äì e.g., true

3. Input & Output

Input lets users enter data; Output shows results.

let name = prompt("What is your name?");
alert("Welcome, " + name);

4. Operators

Operators perform calculations or comparisons:

  • + Add
  • - Subtract
  • * Multiply
  • / Divide
  • == Compare equality

5. Conditional Statements (if...else)

These control what the program does based on conditions:

if (age > 18) {
  alert("You are an adult");
} else {
  alert("You are a minor");
}

6. Loops

Loops repeat tasks automatically.

for (let i = 1; i <= 5; i++) {
  console.log("Count: " + i);
}
