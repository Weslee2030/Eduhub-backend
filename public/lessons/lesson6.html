What is Object-Oriented Programming (OOP)?

OOP is a programming approach that organizes code into objects ‚Äî logical units that contain data (properties) and behavior (methods).

This approach helps to model real-world problems more easily and makes code more reusable, maintainable, and scalable.

üîë Core Concepts of OOP

1. Class

A class is like a blueprint or template. It defines what an object should contain and how it should behave. It doesn‚Äôt do anything on its own ‚Äî we use it to create actual objects.

2. Object

An object is an instance of a class. It holds specific data and can perform functions defined in the class. You can think of a class as a ‚Äútype‚Äù (e.g., Car), and the object as a real car (e.g., a red Toyota made in 2021).

3. Encapsulation

Encapsulation means wrapping data and methods into a single unit (a class), and protecting internal data by using access modifiers (like `private` or `protected`). This keeps the object‚Äôs data safe and prevents direct access from outside the object.

4. Inheritance

Inheritance allows one class (child) to use the properties and methods of another class (parent). This avoids code duplication. For example, a class ElectricCar can inherit from a class Car.

5. Polymorphism

Polymorphism means ‚Äúmany forms.‚Äù The same method or function can behave differently depending on the object calling it. This can be achieved through method overriding (in child classes) or method overloading (with different parameters).

üß† Why is OOP Important?

  • Improves code structure and readability
  • Enables reuse through inheritance
  • Makes maintenance easier
  • Widely used in building large applications
  • Supported by languages like Python, Java, C++, JavaScript, PHP, and C#

üíª Example: OOP in JavaScript

// Defining a class
class Student {
  constructor(name, course) {
    this.name = name;
    this.course = course;
  }

  introduce() {
    return `Hi, I'm ${this.name} and I study ${this.course}`;
  }
}

// Creating objects
const student1 = new Student("Wesley", "Computer Science");
const student2 = new Student("Joy", "Data Science");

console.log(student1.introduce());  // Hi, I'm Wesley and I study Computer Science
console.log(student2.introduce());  // Hi, I'm Joy and I study Data Science
