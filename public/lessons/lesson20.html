üì¶ What is a Database?

A database is a structured collection of data that can be accessed, managed, and updated efficiently. Databases are used in nearly every software application, including websites, mobile apps, and enterprise systems.

üõ†Ô∏è Types of Databases

  • Relational Databases (SQL): Organize data in tables (rows and columns). Example: MySQL, PostgreSQL
  • NoSQL Databases: Use flexible data models like documents or key-value pairs. Example: MongoDB, Firebase

üßë‚Äçüíª What is SQL?

SQL (Structured Query Language) is used to communicate with relational databases. It helps create, read, update, and delete data (often called CRUD operations).

üìã Common SQL Commands

-- Create a table
CREATE TABLE users (
  id INT PRIMARY KEY,
  name VARCHAR(100),
  email VARCHAR(100)
);

-- Insert data
INSERT INTO users (id, name, email) VALUES (1, 'Wesley', 'wesley@example.com');

-- Read data
SELECT * FROM users;

-- Update data
UPDATE users SET name = 'Wesley Orina' WHERE id = 1;

-- Delete data
DELETE FROM users WHERE id = 1;

üîê Why Databases Matter

  • Store and retrieve large amounts of data quickly
  • Ensure data integrity, security, and relationships
  • Allow efficient access by multiple users or systems
