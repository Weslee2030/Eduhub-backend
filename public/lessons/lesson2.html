üìù Learning Notes

  • Data Structures are ways to organize and store data in a computer.
  • Arrays are fixed-size lists of elements stored in memory. Example: [10, 20, 30]
  • Lists are like arrays, but more flexible in some languages.
  • Stacks use the "Last In, First Out" (LIFO) method. Like a pile of plates.
  • Queues use "First In, First Out" (FIFO). Like a line at the bank.
  • Algorithms are step-by-step instructions to solve a problem (e.g., sorting numbers).
  • Efficiency is important ‚Äî fast code saves time and resources.

üíª Code Example

// Simple Array in JavaScript
let numbers = [5, 10, 15];
console.log(numbers[0]); // Outputs: 5

// Stack example
let stack = [];
stack.push("A");
stack.push("B");
console.log(stack.pop()); // Outputs: B

// Queue example
let queue = [];
queue.push("A");
queue.push("B");
console.log(queue.shift()); // Outputs: A
