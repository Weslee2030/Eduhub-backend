What is a Database?

A database is a structured system for storing, managing, and retrieving data efficiently. It allows applications to save and access data reliably.

üîç Types of Databases

1. SQL (Relational Databases)

  • Use tables with rows and columns
  • Structured Query Language (SQL) for managing data
  • Data is organized with schemas (defined structure)
  • Examples: MySQL, PostgreSQL, SQLite, SQL Server

2. NoSQL (Non-relational Databases)

  • Flexible schema or schema-less (good for changing data)
  • Use documents, key-value pairs, graphs, or wide-columns
  • Great for big data and real-time applications
  • Examples: MongoDB, Firebase, CouchDB, Cassandra

üíª SQL Example (MySQL)

-- Create a table
CREATE TABLE users (
  id INT AUTO_INCREMENT,
  name VARCHAR(100),
  email VARCHAR(100),
  PRIMARY KEY(id)
);

-- Insert data
INSERT INTO users (name, email) VALUES ('Wesley', 'wesley@example.com');

-- Fetch data
SELECT * FROM users;

üíª NoSQL Example (MongoDB)

// Insert document
db.users.insertOne({ name: "Wesley", email: "wesley@example.com" });

// Find all users
db.users.find();
