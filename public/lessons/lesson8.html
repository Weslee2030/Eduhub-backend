What Are Algorithms?

An algorithm is a step-by-step procedure or formula for solving a problem. It's like a recipe ‚Äî input goes in, steps are followed, and an output is produced.

Examples of common algorithms:

  • Sorting (e.g., Bubble Sort, Merge Sort)
  • Searching (e.g., Linear Search, Binary Search)
  • Pathfinding (e.g., Dijkstra‚Äôs Algorithm, A*)
  • Recursion (functions calling themselves)

What Are Data Structures?

A data structure is a way to organize, manage, and store data efficiently. It determines how data is accessed and modified.

Examples of data structures:

  • Arrays: List of elements, accessed by index
  • Stacks: LIFO (Last-In, First-Out)
  • Queues: FIFO (First-In, First-Out)
  • Linked Lists: Chain of nodes with pointers
  • Trees: Hierarchical data (e.g., folder system)
  • Graphs: Nodes connected by edges (e.g., maps, social networks)

üß† Why It Matters

  • Algorithms + Data Structures = Efficient Code
  • They are used in all major systems (search engines, apps, games)
  • Key in technical interviews and real-world coding problems

üíª Example: Binary Search in JavaScript

function binarySearch(arr, target) {
  let left = 0;
  let right = arr.length - 1;

  while (left <= right) {
    const mid = Math.floor((left + right) / 2);
    if (arr[mid] === target) return mid;
    else if (arr[mid] < target) left = mid + 1;
    else right = mid - 1;
  }

  return -1;
}

const nums = [1, 3, 5, 7, 9, 11];
console.log(binarySearch(nums, 7)); // Output: 3
