üåê What is Networking?

Networking is the practice of connecting computers and other devices to share data. The Internet is the largest network, connecting billions of devices worldwide.

üì° Key Concepts

  • IP Address: A unique identifier for a device on a network (e.g. 192.168.1.1)
  • DNS: Converts domain names (like google.com) to IP addresses
  • HTTP/HTTPS: Protocols used for web communication
  • Client-Server Model: Client (browser) sends a request, server sends a response
  • API: A way for programs to communicate over the internet

üì¶ How the Internet Works

  1. You type a website name (e.g. eduhub.com)
  2. DNS converts it to an IP address
  3. Your browser sends an HTTP request to that address
  4. The server processes it and sends back an HTML response
  5. Your browser renders the page

üß∞ Tools for Networking

  • ping ‚Äî Check if a server is reachable
  • traceroute ‚Äî See the path data takes to a server
  • curl ‚Äî Test API requests
  • netstat ‚Äî View open ports and active connections
