What is Computer Architecture?

Computer architecture is the internal design of a computer system. It defines how hardware components interact to execute programs and manage data.

1. Central Processing Unit (CPU)

  • The CPU is the "brain" of the computer.
  • It carries out instructions from software programs.
  • ALU (Arithmetic Logic Unit): Handles all calculations and logical decisions.
  • CU (Control Unit): Coordinates operations between memory, input/output, and ALU.
  • Works in a cycle: Fetch ‚Üí Decode ‚Üí Execute

2. Memory (RAM)

  • RAM = Random Access Memory (temporary storage).
  • Stores data while programs are running.
  • RAM is fast but loses all data when power goes off.

3. Storage

  • Hard Disk Drive (HDD): Mechanical, slower but cheaper.
  • Solid State Drive (SSD): Faster and more durable.
  • Used for saving files permanently (e.g., videos, software, documents).

4. Input and Output Devices

  • Input Devices: Send data to the computer (keyboard, mouse, scanner).
  • Output Devices: Show results to users (monitor, printer, speakers).

5. Instruction Cycle

  • Fetch: Get an instruction from memory.
  • Decode: Understand what the instruction means.
  • Execute: Carry out the instruction.
  • This cycle repeats millions of times per second!
