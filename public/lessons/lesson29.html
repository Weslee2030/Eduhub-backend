üß± What are Data Structures?

Data Structures are ways of organizing and storing data efficiently so it can be accessed and modified effectively. Examples include arrays, lists, trees, stacks, queues, graphs, and hash tables.

üìö Common Data Structures

  • Array: Stores elements in a fixed-size sequential list
  • Linked List: Each element points to the next one
  • Stack: Follows LIFO (Last-In-First-Out)
  • Queue: Follows FIFO (First-In-First-Out)
  • Tree: Hierarchical structure (e.g., Binary Trees)
  • Hash Table: Stores key-value pairs for fast access
  • Graph: A collection of nodes and edges

üß† What are Algorithms?

Algorithms are step-by-step procedures for solving problems or performing tasks. Efficiency of an algorithm is measured using time and space complexity.

üöÄ Example Algorithm - Binary Search

function binarySearch(arr, target) {
  let left = 0, right = arr.length - 1;
  while (left <= right) {
    let mid = Math.floor((left + right) / 2);
    if (arr[mid] === target) return mid;
    else if (arr[mid] < target) left = mid + 1;
    else right = mid - 1;
  }
  return -1;
}
