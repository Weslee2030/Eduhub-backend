What is an Operating System (OS)?

An operating system is system software that manages computer hardware and software resources. It acts as a bridge between the user and the computer hardware.

Main Functions of an OS

  • 1. Process Management: Manages running applications (processes), decides which gets CPU time.
  • 2. Memory Management: Allocates and tracks memory (RAM) used by programs.
  • 3. File System Management: Manages data storage (folders, files, permissions).
  • 4. Device Management: Controls hardware like keyboard, mouse, printer, USB, etc.
  • 5. User Interface: Provides CLI (like Linux Terminal) or GUI (like Windows, Android).

Types of Operating Systems

  • Batch OS: Executes jobs in batches with no user interaction.
  • Time-Sharing OS: Multiple users can interact with the system at once (e.g. UNIX).
  • Embedded OS: Found in devices like smartwatches, cars, ATMs.
  • Mobile OS: Android, iOS ‚Äì designed for phones and tablets.
  • Real-Time OS (RTOS): Used in systems needing immediate response (e.g. robotics).

  • Windows: Most common desktop OS
  • Linux: Open-source, used by developers and servers

Why is the OS Important?

  • Without an OS, hardware cannot function.
  • It allows you to run programs, browse the internet, play games, and interact with devices.
  • It keeps your system secure and efficient.
