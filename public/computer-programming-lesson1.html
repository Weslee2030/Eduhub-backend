Computer Programming - Topic 1

Introduction to Programming

Lesson Summary (from video):

  • Programming is the act of writing instructions for a computer to follow.
  • We use programming languages (like Python, Java) to write these instructions.
  • Computers only understand binary (1s and 0s), so programming languages help us write instructions in a way humans can understand.
  • There are different types of programming languages: high-level (like Python) and low-level (closer to machine code).
  • Programs consist of algorithms — step-by-step procedures to solve problems.

Code Example:

# This program prints a message
print("Hello, world!")
